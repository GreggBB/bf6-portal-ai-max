# BF6 Portal AI-MAX SDK

> **AI-optimized toolkit for building Battlefield 6 Portal mods**

Build Portal mods faster using AI coding assistants like Claude, Gemini, Cursor, Windsurf, and others.

## Features

- **35+ Verified Patterns** - Tested code patterns covering the entire Portal SDK
- **Multi-AI Support** - Instructions for Claude, Gemini, and generic AI agents
- **Process Workflows** - Step-by-step guides for discovery, planning, building, and debugging
- **Community Utilities** - 11 helper modules (Events, Timers, Logger, SolidUI, Raycast, etc.)
- **Spatial Generator** - Programmatic tools for creating map layouts

## Quick Start

1. Clone this repository
2. Open in your AI coding environment
3. Tell your AI: *"Read CLAUDE.md to understand how to build Portal mods"*
4. Start building: *"Create a gun game mod"*

## Critical Rules

### 1. BF2042 vs BF6 Isolation
BF2042 was internally "Battlefield 6" - code looks similar but is NOT compatible.
**Only use patterns from this SDK, never web-sourced BF2042 examples.**

### 2. Object Comparison
SDK objects require `GetObjId()` for comparison:

```typescript
// WRONG
if (team1 === team2) { }

// CORRECT
if (mod.GetObjId(team1) === mod.GetObjId(team2)) { }
```

### 3. Function Validation
AI models may invent plausible-sounding functions that don't exist.
Always validate `mod.*` calls against the pattern library.
